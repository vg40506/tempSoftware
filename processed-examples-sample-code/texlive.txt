Use Case: LaTeX document preparation system

Code details and examples:

Input file (example.tex):
```latex
\documentclass{article}
\begin{document}
Hello, this is a sample LaTeX document.
\end{document}
```

Command to compile the LaTeX file:
```
pdflatex example.tex
```

This will generate a PDF file "example.pdf" from the LaTeX source file "example.tex".
