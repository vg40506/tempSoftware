Use Case: libogg is a library for encoding and decoding Ogg format files.
Code details and examples: 
1. Encode a WAV file to Ogg format:
Sample input file (input.wav):
```
Input file: input.wav
```

Code snippet:
```bash
oggenc input.wav -o output.ogg
```

2. Decode an Ogg file to WAV format:
Sample input file (input.ogg):
```
Input file: input.ogg
```

Code snippet:
```bash
oggdec input.ogg -o output.wav
```
