Use Case: Use GCC for compiling C code
Code details and examples:

Input file (example.c):
```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

Command to compile using GCC:
```
gcc example.c -o example
```

Command to run the compiled executable:
```
./example
```
